The framework

The coaching loop:
coach, check, adjust.

Most coaching dies because nobody checks the next call. The whole method is four turns: coach one thing, watch the next call, ask did it stick?, then adjust. Works on a whiteboard. Works better when AI runs it for you. Steal the loop either way.

How to talk to Claude

Four moves that get
Claude's best work.

🎭

Give it a role

"You're my coaching log for Jordan." Claude writes sharper when it knows who it's being.

📎

Show, don't tell

Paste a real call, not a description of one. Examples beat adjectives every single time.

🧠

Give it memory

One running file per rep, pasted back each time, so nothing resets every Monday.

🔁

Make it self-critique

"What's the weakest part of that?" then "now fix it." Two prompts, far better output.

Step 1, keep a file per rep

The non-obvious move: don't paste a call once. Keep one running markdown file per rep so Claude has memory you control.

C:\reps\> dir rep-jordan.md rep-priya.md rep-marcus.md C:\reps\> claude open rep-jordan.md + today's call ▌

Step 2, the prompt that does the real work

Paste the rep's log plus the new call. Ask Claude to check whether last call's coaching showed up.

You keep a coaching log for one of my reps. I'll paste the log, then a new call. Update it: 1. Add a dated entry, 2 lines. 2. Take the LAST "next focus." Did it show up in THIS call? Quote the moment, or say it didn't. 3. Give ONE specific focus for next call. 4. Update a "patterns" list up top.
# rep-jordan.md · patterns: strong rapport, weak on budget Apr 18, Northwind last focus "ask budget early", slipped ✕

Step 3, render it into a scorecard

Hand the markdown back and ask for a one-file HTML scorecard. Coaching becomes glanceable.

Take this log and build a single self-contained HTML scorecard: rep + current focus, last 5 calls showing focus stuck (green) or slipped (red), top recurring gap.

Bonus, a 60-second pre-call game plan

Before the call, not after. Most reps wing it; this is the edge.

Research [company] + [contact]. Give me a one-page game plan: their likely top priority, 3 discovery questions that open budget, the objection I'll hit, and the ONE trap to avoid on this call.

Bonus, practice on Claude, not the customer

Reps get reps. Claude plays the skeptic so the real call isn't the first try.

Be a skeptical [VP of Sales] who already uses a competitor. I'll pitch. Push back the way they really would. After 5 exchanges, score my discovery, objection handling, and next-step.
